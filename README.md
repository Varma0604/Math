# Math Adventure for Kids

## Project Description

**Math Adventure for Kids** is an interactive web-based learning game designed to make math fun and engaging for children. The game provides a variety of math problems that challenge kids to solve equations while enjoying an animated and colorful environment. With exciting avatars, immediate feedback, and a leaderboard, kids can learn math concepts while competing with friends and tracking their progress.

## Features

- **Dynamic Math Problems**: Generates random addition, subtraction, and multiplication problems.
- **Immediate Feedback**: Provides instant feedback on answers, encouraging learning through positive reinforcement.
- **Leaderboard**: Displays scores for multiple players to foster a friendly competitive atmosphere.
- **Colorful Design**: Engaging animations and vibrant backgrounds create an inviting experience.
- **Avatar Customization**: Users earn different avatars based on their scores, making the game more personalized.
- **Progress Tracking**: Visual progress indicators help kids see their improvement over time.
